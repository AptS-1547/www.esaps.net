## 从0走入Hello Python - 本地版
> 环境一定要安装好！一定要安装官方的！不要去淘宝花39元买安装服务！         --AptS:1547

为了达到高效写代码的目标，我们需要安装两个软件：`Python` 以及 `IDE（也称为集成开发和学习环境）`

Python就是用来执行你的代码的软件，而IDE给你提供了高度集成的开发环境，方便我们进行语言的学习

### 安装教程：Python

共有3种方式在Windows上安装Python
 - 进入Python官网下载安装包：[python.org](https://www.python.org/downloads/)
 - 点击这条链接下载Python 3.12.7：[Python 3.12.7 For Windows (64-bit)](https://www.python.org/ftp/python/3.12.7/python-3.12.7-amd64.exe)
 - 在Microsoft Store搜索Python安装

**不论何种方式安装，Python安装版本建议至少为3.6.5以上**

对于前两种安装方式，我们需要自行安装并进行安装配置操作
