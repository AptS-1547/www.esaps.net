## 如何申请 GitHub Copilot

GitHub Copilot 是一个由 GitHub 和 OpenAI 联合开发的 AI 编程助手，可以帮助你更快地编写代码。以下是申请和使用 GitHub Copilot 的步骤：

## 1. 注册 GitHub 账号

如果你还没有 GitHub 账号，请先前往 [https://github.com/signup](https://github.com/signup) 注册一个账号。

## 2. 设置Github账号

我们可以通过 GitHub Education 计划免费获取 GitHub Copilot。但在之前，我们需要做一些准备工作。

### 2.1 设置Github账号信息

1. 登录你的 GitHub 账号。
2. 打开[Github 账号邮箱界面](https://github.com/settings/emails)，给我们的Github账号添加学校给我们的邮箱地址
  `（例如：apts-1547@example.com）`

3. 打开[Github Profile设置界面](https://github.com/settings/profile)，按照下图所示设置你的账号信息
![](https://www.esaps.net/wp-content/uploads/2025/03/391742175340_.pic_.webp)
4. 打开[Github 账单设置界面](https://github.com/settings/billing/payment_information)，按照下图所示设置你的账单信息
![](https://www.esaps.net/wp-content/uploads/2025/03/401742175367_.pic_.webp)
5. 打开[Github 账号安全设置界面](https://github.com/settings/security)，按照下图所示设置账号二步验证
![](https://www.esaps.net/wp-content/uploads/2025/01/image-7.webp)
  接下来会让你下载恢复密钥，下载后保存好，点击下一步，即完成设置

### 2.2 获取在校就读证明

在此步骤中，你需要获得一份在校就读证明，证明你是一名在校学生。
证明可以是学生证、学校官方邮件或者学校官方网站上的学生信息页面截图。
请确保证明上包含你的姓名、学校名称和在校时间。
你可以在你的学校官网上找到这些信息，或者联系学校的教务处或学生处获取。

## 3. 申请 GitHub Education

1. 前往 [GitHub Education 主页](https://github.com/education)。

2. 点击页面上的 **`Join Github Education`** 按钮，进入提交申请界面，选择自己所在的学校并点击 **`Continue`** 按钮。
![](https://www.esaps.net/wp-content/uploads/2025/01/image-2.webp)
![](https://www.esaps.net/wp-content/uploads/2025/01/image-12.webp)
  **注意：在此过程中请勿使用任何网络加速工具！确保你的账号信息已经设置完成，否则会影响申请结果**、

3. 在下一步，Github 需要获取你的位置信息以及打开摄像头要求你拍照相关在校证明，如下图
![](https://www.esaps.net/wp-content/uploads/2025/01/image-14.webp)
  点击 **`允许`** 、**`Take a picture`** 、 **`允许`** 并拍照上传证明
  如果电脑没有摄像头或者拍照模糊，可以使用手机拍照上传，方法和上面一样

4. 上传在校就读证明，点击 **`Process my application`** 按钮提交申请。

5. 等待审核结果，一般情况下会在 1~2 分钟内收到审核结果。
![](https://www.esaps.net/wp-content/uploads/2025/01/image-18.webp)
![](https://www.esaps.net/wp-content/uploads/2025/01/image-19.webp)

目前的审核通过后大约需要9天左右权益才会发放，审核通过后，你将会收到一封邮件，邮件中会包含你的权益信息，包括 GitHub Copilot 的使用权益。

## 4. 安装 GitHub Copilot 插件

GitHub Copilot 目前支持多种编辑器，包括 Visual Studio Code、Neovim 和 JetBrains 系列。以下是安装 Visual Studio Code 插件的步骤：

1. 打开 Visual Studio Code。
2. 进入扩展市场（左侧栏的方块图标）。
3. 搜索 **"GitHub Copilot"**。
4. 点击 **"Install"** 按钮安装插件。

## 5. 配置 GitHub Copilot

安装完成后，你可以在 VSCode 上方栏目发现一个小机器人图标，您可以点击此图标进入聊天界面
第一次使用需要您登录 Github 账号，请按照他的指引进行登录

注意：部分情况发现你需要在Github官网手动设置Copilot的权限，具体操作如下：

1. 打开[Github Copilot设置界面](https://github.com/settings/copilot)，一路下一步就好了

## 6. 开始使用 GitHub Copilot

1. 打开一个代码文件。
2. 开始编写代码，GitHub Copilot 会自动提供代码建议。
3. 使用 `Tab` 键接受建议，或使用 `Esc` 键忽略建议。
4. 或者，你可以在栏目条右方点击小机器人图标，进入聊天模式

## 常见问题

- **如何禁用 GitHub Copilot？**

  - 你可以在 Visual Studio Code 的扩展市场中禁用或卸载 GitHub Copilot 插件。

- **GitHub Copilot 是否支持所有编程语言？**
  - GitHub Copilot 支持多种编程语言，但对一些语言的支持可能更好。
